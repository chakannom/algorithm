========================================================================
    콘솔 응용 프로그램 : Cohen-Sutherland Line Clipping Algorithm 프로젝트 개요
========================================================================

응용 프로그램 마법사에서 이 Cohen-Sutherland Line Clipping Algorithm 응용 프로그램을 만들었습니다.  

이 파일에는 Cohen-Sutherland Line Clipping Algorithm 응용 프로그램을 구성하는 각 파일에 대한
요약 설명이 포함되어 있습니다.


Cohen-Sutherland Line Clipping Algorithm.vcproj
    응용 프로그램 마법사를 사용하여 생성한 VC++ 프로젝트의 기본 프로젝트 파일입니다. 
    파일을 생성한 Visual C++ 버전에 대한 정보와 응용 프로그램 마법사를 사용하여 선택한
    플랫폼, 구성 및 프로젝트 기능에 대한 정보가 포함되어 있습니다.

Cohen-Sutherland Line Clipping Algorithm.cpp
    기본 응용 프로그램 소스 파일입니다.

/////////////////////////////////////////////////////////////////////////////
기타 표준 파일:

StdAfx.h, StdAfx.cpp
    이 파일은 미리 컴파일된 헤더(PCH) 파일인 Cohen-Sutherland Line Clipping Algorithm.pch와
    미리 컴파일된 형식(PCT) 파일인 StdAfx.obj를 빌드하는 데 사용됩니다.
